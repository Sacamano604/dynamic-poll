Dynamic Poll

Another small project I've completed in AngularJS. This polling system allows you to create a poll, share a link, vote on it and see the results.

Some of the details are:

  • Create a poll which stores the poll name and options in a mysql databse
  • Once the poll is created you can copy your link to share it with others
  • Uses php sessions so you may only vote once
  • Uses angular's interval to fetch the db results every two seconds so you can see poll results in near real-time

Future goals:

  • Implement with firebase for real-time poll tracking, a lot cleaner than querying the databse every two seconds
  • Restucture data so I can sort the poll's columns, and put the most voted item on the top of the list
